ARMORS Control Panel Backend
Armors Control Panel backend

This is python backend for Armors Control Panel application.

Development requirements

  1. Python 3.10 or above

Configuration

When configuring project for the first time, create your python virtual environment:

cd ./backend
python -m venv .venv

Then activate it (you will need to do it always when you want to use it):

./.venv/Scripts/Activate.ps1

Install project with dependencies:

pip install -r ./requirements.txt
pip install -e .

You can use make command for predefined commands (you will need to have GNU make installed on your system):

make

For all commands see Makefile

Running and testing

To start backend application run:

python ./acp/main.py

Contributing

When you want to add new functionality make sure that static analysis and linting doesn't return any errors. To check for errors run:

python -W ignore -m autoflake --in-place --recursive --ignore-init-module-imports --remove-duplicate-keys --remove-unused-variables --remove-all-unused-imports .
python -m black .
python -m isort .
python -m mypy . --ignore-missing-imports --exclude 'build/*'

or shorter:

make lint
